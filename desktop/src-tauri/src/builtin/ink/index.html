卷首 · 一篇

旧书摊的黄昏

收摊前的半小时，是旧书摊最诚实的时候：白天的过客散尽了， 还蹲在摊前的人，是真的想带一本书回家。这一篇记城南的老陈、 一册一九八二年的《草木小记》，和扉页上那两行淡得快要退回纸里去的铅笔字。

丙午 · 二〇二六

  • 七月十四 旧书摊的黄昏 收摊前的半小时，是旧书摊最诚实的时候。
  • 六月三十 晚风记 晚饭后的一条河堤，走了七年，还没走完。
  • 六月十一 祖父的钢笔 笔尖磨成了偏锋，写出来的字却比我的正。
  • 五月廿三 深夜面馆 凌晨两点的一碗光面，比什么都更像家。
  • 五月初七 为「无用」一辩 阳台上那盆不结果的金桔，替我挡住了许多着急。
  • 四月十九 巷口修表铺 他修的不是表，是别人不肯扔的时间。
  • 三月廿八 搬家记 三百本书搬了四趟，扔掉的只有一把坏伞。

乙巳 · 二〇二五

  • 十二月十二 冬至前后 甜是小城冬天的方言。
  • 十一月初九 雨天读帖 窗外下它的雨，纸上下它的雨。
  • 十月十七 夜航船 慢船比快船多出来的，正是路上那一段。

更早的文字散在旧笔记本里，抄清一篇，收一篇。

又记

沈砚秋，教书为业，居南方小城。白天讲课，晚上写字；写得慢，改得多， 所以这册手帖更新不快——但凡挂出来的，都是改到自己不脸红的。

「墨白」二字取意于纸上两件事：落墨要省，留白要舍得。 通信处在 yanqiu@example.com， 来信必复，只是慢。
